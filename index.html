CheatSheetGit

A CheatSheet For Git Versionning

Create Local Change Commit History Update & Publish Branch & tags Merge & Rebase Undo
CREATE
Clone an existing repository
$ git clone ssh user@domaine.com/repository.git
Create a new local repository
$ git init
LOCAL CHANGES
Changed files in working directory
$ git status
Change to track files
$ git diff
Add current change to next commit
$ git add files
Commit prev. staged changes with message
$ git commit -m "message"
COMMIT HSTORY
Show all commits
$ git log
Who change what and when in file
$ git blame file
UPDATE & PUBLISH
List all currently configured remotes
$ git remote -V
Show information about a remote
$ git remote show remote
Add new remote repository named remote
$ git remote add shortname url
Download changes and merge to HEAD
$ git pull remote branch
Publish local change on a remote
$ git push remote branch
BRANCH & TAGS
List all existing branch
$ git branch -av
Switch HEAD branch
$ git checkut branch
Create a new branch
$ git branch branch
Delete a local branch
$ git branch -d branch
MERGE & REBASE
Merge branchinto your HEAD
$ git merge branch
Rebase your current HEAD onto branch
$ git rebase branch
Use merge tool to resolve conflict
$ git mergetool
UNDO
Discard all local change in working directory
$ git reset --hard HEAD
Discard local change in file
$git checkout HEAD file
Revert a commit
$ git revert commit
